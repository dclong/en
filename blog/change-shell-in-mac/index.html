Change Shell in Mac

Apr 02, 2018

You can check the current shell in use by running the following command.

echo $SHELL

However, notice that echo $SHELL does not report the right shell sometimes. A more robust way is to use echo $0.

The default shell can be changed using thee comamnd chsh. For example, thee following command changes the default shell to Bash.

chsh -s /bin/bash

Notice that Hyper does not honor the the environment variable SHELL and thus the above way of changing the default shell does not work for Hyper.js. One way to change the default shell in Hyper.js is to directly update the field shell in the configuration file ~/.hyper.js to be the desire one.
